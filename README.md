# Getting Started.

### Requirements
+ Docker 20.10.7+
+ Docker-compose 1.29.2+

### Execute application.

run docker command.

```shell
docker build -t webapp .
```

After build, you must execute the next command.

```shell
docker-compose up
```

### Sagger ui
After run the application, go to swagger documentation: http://localhost:8080/swagger-ui.html
